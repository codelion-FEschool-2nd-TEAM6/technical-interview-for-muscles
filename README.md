# 프론트엔드 기술 면접 스터디

1. 시행: 매 주 금요일 게더타운 오전 10시
2. 방식
- 매 주 수요일까지 각자 맡은 파트를 md파일에 정리하여 레포지토리에 푸쉬합니다. (가급적 수요일까지 엄수 부탁드립니다!)
- 스터디 전까지 다른 팀원들이 올린 파트 파일들을 보고 공부해옵니다.
- 금요일 스터디 당일 자신이 맡은 파트를 제외한 나머지 파트들 중 하나를 사다리타기로 랜덤으로 정하여 발표합니다.

- 태희: SSR, CSR
